# Hate-speech-detector-
what is hate speech? 
<br>
any speech that disparage a group of people because of their race, religion, nationality, national origin, sexual orientation, or gender identity is considered hate speech 
<br>
for the purpose of the NLP task I have used this dataset https://www.kaggle.com/datasets/mrmorj/hate-speech-and-offensive-language-dataset from kaggle
<br>
This is the dataset which contain tweets that are either offensive or posts/comments have derogoratory, sexist, hate spceech content
